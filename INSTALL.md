# Adversarial Coder - Skill Package

This directory contains the OpenClaw skill for adversarial coding.

## Files

- `SKILL.md` - OpenClaw skill definition (metadata + description)
- `adversarial-coder` - Main executable script
- `README.md` - User documentation

## Installation

Copy this directory to your OpenClaw skills folder:

```bash
cp -r . ~/.nvm/versions/node/v24.14.0/lib/node_modules/openclaw/skills/adversarial-coder/
```

Or install via ClawHub (when available):

```bash
clawhub install adversarial-coder
```

## Usage

```bash
adversarial-coder "Your prompt here"
```

See README.md for full documentation.
